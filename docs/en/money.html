For Facebook platforms, spending data from the API does not provide aggregate statistics per advertiser. Instead, it only provides very broad boundaries of money spent. For this reason, the data displayed here is from the Facebook Ad Library Report which does provide money spent by advertiser and is retrieved daily. However, one caveat is that if a perty spent below 100 Euros in a day it will not show the exact amount. For this reason we show confidence intervals around the amount.
